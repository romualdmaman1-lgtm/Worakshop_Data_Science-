Variable-by-variable decisions

age

  • Default summary: median (IQR) — keeps the default. Age ranges from 24–37 and is roughly symmetric, so median (IQR) is perfectly reasonable.
  • Default test: Wilcoxon rank-sum test — override to t.test. Age is roughly normally distributed (bell-shaped), so a t-test is slightly more powerful here. Both tests will agree, but t-test matches the mean if we wanted that.

days_since_last_sex

  • Default summary: median (IQR) — keep default. This variable is heavily right-skewed (most values 1–14, outliers up to 48). Median (IQR) is the right choice for skewed data.
  • Default test: Wilcoxon rank-sum test — keep default. Wilcoxon is non‑parametric and doesn’t assume normality, which is ideal for skewed data.

birth_control

  • Default summary: n (%) — keep default. Two categories (Depoprovera vs no hormonal birth control), nicely balanced (25 vs 26).
  • Default test: Chi-squared test — keep default. All expected counts are well above 5, so chi-squared is fine.

education

  • Default summary: n (%) — keep default. Four ordered categories, but in a standard Table 1 they’re usually presented as counts with percentages.
  • Default test: Chi-squared test — override to fisher.test. One of the categories (e.g., less than grade 9) has only 9 people total, so some cells may have small expected counts. Fisher’s exact test is more reliable in that situation.

pid (participant ID)

  • Exclude from the table — it’s an identifier, not a characteristic.

sex

  • Exclude — all 51 participants are female. A constant column adds no information and can’t be tested.

Build the Table 1

table_one <- metadata |>
  select(-pid, -sex) |>                  # drop identifier and constant column
  tbl_summary(
    by = arm,                             # compare yogurt vs unchanged_diet
    statistic = list(
      all_continuous() ~ "{median} ({p25}, {p75})",
      all_categorical() ~ "{n} ({p}%)"
    ),
    digits = list(
      all_continuous() ~ 1,
      all_categorical() ~ c(0, 1)
    ),
    label = list(
      days_since_last_sex ~ "Days since last sex",
      birth_control ~ "Birth control method"
    ),
    missing_text = "Missing"
  ) |>
  add_p(
    test = list(
      all_continuous() ~ "wilcox.test",
      age ~ "t.test",                      # override: use t-test for age
      education ~ "fisher.test"            # override: use Fisher's for education
    ),
    pvalue_fun = ~ style_pvalue(.x, digits = 2)
  ) |>
  add_overall() |>                         # add an "Overall" column
  bold_labels() |>
  modify_header(label = "**Characteristic**")

Antibody analysis — Maternal vs Cord blood by HIV status and child sex

Paired comparison of CMV, RSV, Enterovirus, and TT antibody levels

Overview

This notebook compares maternal blood antibody levels vs cord blood antibody levels for four pathogens: CMV, RSV, Enterovirus, and TT (Tetanus Toxoid).

  • Paired analysis — each mother-cord pair comes from the same dyad
  • Stratified by HIV status (Negative / Positive) and child sex (F / M)
  • Wilcoxon signed-rank test (non-parametric paired test, chosen because differences are not normally distributed)
  • All figures and labels are in English
library(tidyverse)
library(here)
library(ggpubr)
library(patchwork)

donnees <- read_tsv(here("data/etude_mere_enfant.csv"), na = c("", "NA", "f"))
donnees <- donnees %>%
  mutate(`CMV cord` = as.numeric(`CMV cord`))
# Reshape to long format for faceted plotting
antibody_pairs <- list(
  "CMV" = list(mother = "CMV mother", cord = "CMV cord"),
  "RSV" = list(mother = "RSV mother", cord = "RSV cord"),
  "Enterovirus" = list(mother = "Enterovirus mother", cord = "Enterovirus cord"),
  "TT" = list(mother = "TT mother", cord = "TT Cord")
)

# Build a long dataset for plotting
long_data <- bind_rows(lapply(names(antibody_pairs), function(nm) {
  m_col <- antibody_pairs[[nm]]$mother
  c_col <- antibody_pairs[[nm]]$cord
  donnees %>%
    select(`HIV STATUS`, `child SEX`, mother = all_of(m_col), cord = all_of(c_col)) %>%
    mutate(antibody = nm, .before = 1)
})) %>%
  filter(!is.na(mother), !is.na(cord), !is.na(`child SEX`)) %>%
  # Create a paired ID within each antibody group
  group_by(antibody) %>%
  mutate(pair_id = row_number()) %>%
  ungroup() %>%
  # Pivot to long format for ggplot
  pivot_longer(cols = c(mother, cord), names_to = "source", values_to = "value") %>%
  mutate(source = recode(source, mother = "Maternal blood", cord = "Cord blood"),
         source = factor(source, levels = c("Maternal blood", "Cord blood")),
         `child SEX` = recode(`child SEX`, F = "Female", M = "Male"))

1️⃣ CMV — Cytomegalovirus

# Paired Wilcoxon tests by HIV status and child sex
cmv <- long_data %>% filter(antibody == "CMV")

cmv_results <- expand_grid(
  hiv = c("Negative", "Positive"),
  sex = c("Female", "Male")
) %>%
  rowwise() %>%
  mutate(
    n_pairs = {
      df <- cmv %>% filter(`HIV STATUS` == hiv, `child SEX` == sex, source == "Maternal blood")
      nrow(df)
    },
    median_mother = {
      df <- cmv %>% filter(`HIV STATUS` == hiv, `child SEX` == sex, source == "Maternal blood")
      median(df$value, na.rm = TRUE)
    },
    median_cord = {
      df <- cmv %>% filter(`HIV STATUS` == hiv, `child SEX` == sex, source == "Cord blood")
      median(df$value, na.rm = TRUE)
    },
    wilcox_p = {
      mat <- cmv %>% filter(`HIV STATUS` == hiv, `child SEX` == sex, source == "Maternal blood") %>% pull(value)
      cord <- cmv %>% filter(`HIV STATUS` == hiv, `child SEX` == sex, source == "Cord blood") %>% pull(value)
      if (length(mat) >= 3 && length(cord) >= 3) {
        wilcox.test(mat, cord, paired = TRUE)$p.value
      } else {
        NA
      }
    }
  ) %>%
  ungroup()

knitr::kable(cmv_results %>% rename(HIV = hiv, Sex = sex, N = n_pairs,
                                    `Median mother` = median_mother,
                                    `Median cord` = median_cord,
                                    `p-value` = wilcox_p),
             digits = c(0, 0, 0, 1, 1, 4))
HIV Sex N Median mother Median cord p-value
Negative Female 39 1322622 1307273 0.6737
Negative Male 33 981131 4562208 0.0008
Positive Female 36 2668402 3899910 0.0041
Positive Male 29 1202731 1661928 0.2560 
# Paired dot plot with lines
cmv_long <- cmv %>%
  mutate(pair_id = as.character(pair_id))

ggplot(cmv_long, aes(x = source, y = value, group = pair_id)) +
  geom_line(aes(color = pair_id), alpha = 0.25, show.legend = FALSE) +
  geom_point(aes(fill = source), size = 2.5, shape = 21, color = "black") +
  facet_grid(`child SEX` ~ `HIV STATUS`, scales = "free_y") +
  scale_fill_manual(values = c("Maternal blood" = "#E74C3C", "Cord blood" = "#3498DB")) +
  scale_y_log10(labels = scales::label_comma()) +
  stat_compare_means(
    data = cmv_long,
    aes(x = source, y = value, group = source),
    method = "wilcox.test",
    paired = TRUE,
    label = "p.format",
    label.x = 1.5,
    label.y.npc = "top",
    size = 3.5
  ) +
  labs(
    title = "CMV: Maternal blood vs Cord blood (paired)",
    subtitle = "Each line = one mother-child pair. Log scale.",
    x = "",
    y = "CMV antibody levels"
  ) +
  theme_bw() +
  theme(legend.position = "none",
        strip.text = element_text(face = "bold", size = 11))

CMV finding: Cord blood CMV levels tend to be higher than maternal blood in most subgroups, indicating active transplacental transfer.

7️⃣ Interpretation summary

Antibody Cord vs Mother Pattern Notes
CMV Cord ↑ Mother ↓ Cord higher in most subgroups Variable; depends on maternal serostatus and gestational age at infection
RSV Cord ↑↑ Mother Strongly higher in cord Universal transfer — nearly all cord values exceed maternal values
Enterovirus Cord ↑↑ Mother Strongly higher in cord Very efficient placental transfer
TT Mother ↑ Cord Mother higher than cord Weak transfer — consistent with vaccine-induced IgG kinetics

Key observations

  1. RSV and Enterovirus show the most efficient placental transfer (cord > mother in all subgroups)
  2. TT is the only antibody where maternal levels exceed cord levels — expected for a vaccine antigen
  3. HIV status does not dramatically alter the mother-cord relationship for most antibodies
  4. Child sex effects are subtle — no consistent male/female difference across all antibodies
  5. All comparisons are paired — each mother serves as her own control, giving more statistical power
